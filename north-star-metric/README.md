# north-star-metric

## What it does
Defines a North Star Metric and 3–5 supporting input metrics for any business, classifying the business game (Attention, Transaction, or Productivity) and validating the chosen metric against seven effectiveness criteria. Best used when a founder, PM, or team wants to align on a single customer-centric KPI and produce a shareable framework document.

## Reference files
None.

## Install
Copy this folder into your skills directory, then toggle the skill on in **Customize > Skills**.

## What to change
None. The skill gathers all required context from the user via two structured gates before producing output.

## Usage
**Trigger:** "help me choose our North Star Metric"
**Output:** Polished PDF report saved to [your output folder], containing the business game classification, NSM with 7-criteria validation table, and 3–5 input metrics in a card layout.
