# gtm-strategy

## What it does
Builds a structured go-to-market strategy for a product launch or new market entry. Best used when planning a GTM from scratch, defining a channel mix, or setting KPIs and a launch timeline for a specific product and segment.

## Reference files
None.

## Install
Copy this folder into your skills directory, then toggle the skill on in **Customize > Skills**.

## What to change
- Target segment and product description — supplied by the user at runtime (Gate 1)
- Launch timeline and budget envelope — supplied by the user at runtime (Gate 1)

## Usage
**Trigger:** "Build a GTM strategy for [product]"
**Output:** DOCX strategy document covering channel mix, messaging framework, KPI targets, launch timeline, and a 90-day execution roadmap
