# swot-analysis

## What it does
Runs a structured SWOT analysis for any product, business, startup, or strategic decision. Produces a formatted PDF with a four-quadrant matrix, SO/ST/WO/WT cross-reference insights, and 3–5 prioritised strategic recommendations. Best used for strategic reviews, competitive positioning, market entry evaluation, or board prep.

## Reference files
None.

## Install
Copy this folder into your skills directory, then toggle the skill on in **Customize > Skills**.

## What to change
None. All context is gathered from the user at runtime via Gate 1.

## Usage
**Trigger:** "Run a SWOT analysis on [subject]"
**Output:** PDF report saved to [your output folder]
